AngularJS + Cloud Endpoint

Sample application for using AngularJS backed with a Cloud Endpoint on the Google App Engine.

Client

  • AngularJS 1.2.0-rc.2, HTML enhanced for web apps.
    • angular-resource, support for interacting with RESTful server-side data sources.
    • ui-router 0.2.0, state based and nested routing from the Angular UI package.
    • angular-scenario, E2E testing. Simulate user interaction with the client application.
  • Boostrap 3.0.0, front end framework for faster and easier web development.
  • Moment.js, date and time parsing and formatting library.
  • Jasmine 1.3.1, behavior driven test framework. Uses angular-mocks to handle testing of AngularJS modules.

App Engine

  • Python SDK 1.8.4, application platform.
    • Python NDB, schemaless object datastore.
    • Cloud Endpoints, easily create a web service API.
    • Endpoints Proto Datastore, allows an NDB model to be used directly with a Cloud Endpoints API.
    • Search API, indexed search of generic documents. Supports partial matching of string fields.
